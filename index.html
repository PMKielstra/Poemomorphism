Initial poetry prompts by Chell Stein.

Poemomorphism is digital Consequences.

The first player creates a game. Everybody else joins with their game code.

Everybody writes a line of poetry. Everybody then sees someone else's line and has to continue the poem without context. Repeat until each poem has as many lines as there are players.

If you want more than one line per poem per player, you can set that when creating the game.
